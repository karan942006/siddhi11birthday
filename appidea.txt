🔥 **MASTER PROMPT: BUILD COMPLETE BIRTHDAY SURPRISE ANDROID APP (KOTLIN + FIREBASE)**

You are an expert Android developer, UI/UX designer, and system architect. Build a complete, production-ready Android application named **"SurpriseVerse"**, a birthday surprise app for friends and groups. The app must be fully functional, modern, bug-free, and require no additional changes after generation.

---

# 🎯 CORE REQUIREMENTS

Develop a **fully working Android app** using:

* Language: Kotlin
* UI: Jetpack Compose (modern UI, no XML)
* Architecture: MVVM + Clean Architecture
* Backend: Firebase (Authentication + Firestore + Storage)
* Navigation: Jetpack Navigation Compose
* Async: Kotlin Coroutines + Flow
* Image Loading: Coil
* Animations: Lottie
* State Management: ViewModel + StateFlow

The app must compile and run without errors.

Create full authentication:

* Login & Register

Store user data in Firestore:

* role
